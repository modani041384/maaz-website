Classic platform

Studio

MaaZ Classic Platform Studio provides customers with a comprehensive tool stack for AUTOSAR Classic, with “Configuration, Validation, Generation” as the key value. Our Classic Platform Studio improves your AUTOSAR efficiency, saves time and lowers total cost with easy-to-use tools and features:
  • Based on AUTOSAR standard compliance: Support new AUTOSAR release.
  • User-friendly features.
  • Dynamic for seamless integration with development ecosystem, easy to expand or add more features (Eclipse Base).
  • Support multiple platform with “Configuration, Validation, Generation” features, accelerating microcontroller abstraction layer (MCAL) development.

MaaZ Architecture

MaaZ architecture is built based on the Eclipse ecosystem, covering the entire spectrum of AUTOSAR architecture.
  • Usable both from Eclipse IDE and command-line.
  • EMF as a common meta-model.
  • Transformations implemented using Xtend language.
  • Additional tools and frameworks: maven/Tycho, Junit.
  • Artop is available only to AUTOSAR members.
