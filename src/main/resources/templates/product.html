MaaZ Classic AUTOSAR Software Platform provides an advanced solution for the application software and ECU hardware. This reliable platform enables application software to be largely independent of any MCU and OEM, enabling its reuse for a variety of different ECU systems. With its highly reliable security and functional safety ASIL (Automotive Safety Integrity Level), MaaZ Classic Platform improves quality, maximizes efficiency, and shortens the controller development period.

MaaZ Adaptive AUTOSAR Platform is the solution for E/E architecture, built on high-performance controllers and based on the Adaptive AUTOSAR, providing support for interconnectivity and real-time updates. It provides the AUTOSAR Runtime environment for Adaptive Applications (ARA) and comprises an efficient development environment integrated with Eclipse. MaaZ Adaptive is applied to high-performance ECUs, such as in-vehicle application servers, automated driving, and infotainment systems.
